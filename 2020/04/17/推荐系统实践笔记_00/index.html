README

《推荐系统实践》一书共八个章节，核心章节为第二至六章，文章展开遵循推荐系统架构搭建的具体步骤：选择算法（第二章），处理冷启动问题（第三章），将用户标签、上下文信息和社交网络数据用于算法优化（分别对应第四、五、六章）。

值得注意的是，作者在第二章介绍了三种设计推荐算法的思想，共计四种具体算法。笔记并未按照作者的思路同时搭建四种具体算法的架构，而是分别对各算法的实现进行梳理，依次为：基于用户的协同过滤算法，基于物品的协同过滤算法，隐语义算法和基于图的算法。如果涉及算法间的比较，通常将该部分置于后梳理笔记之中。

非核心章节将独立以章节名进行内容梳理。

关于推荐系统的感想和疑问将会另建笔记整理。

感谢您可以抽出时间阅读《README》，希望这部分内容对你有所价值。
